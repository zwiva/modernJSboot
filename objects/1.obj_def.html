Object:

  • Allows us to store related pieces of information in a single value.
  • All is related a single entity.

 Examples:

user: name, age, email, pass

book: numpages, author, editorial

 Book example

1. Define a variable:

 let myBook = {
  title: 'The Martian Chronicles',
  author: 'Ray Bradbury',
  year: 1950,
  pagecount: 224
 }

2. Console

console.log(myBook)

shows:
{
 title: 'The Martian Chronicles',
 author: 'Ray Bradbury',
 year: 1950,
 pagecount: 224
}

console.property !!!

 console.log(myBook.title)

  shows title: The Martian Chronicles

using template strings:

 console.log(`${myBook.title} author is ${myBook.author}`)

  shows: The Martian Chronicles author is Ray Bradbury

Plus:

  • Type &+nbsp;   (inspect) to add a single space
  • Type &+ensp;   (inspect) to add 2 spaces
  • Type &+emsp;   (inspect) to add 4 spaces
  • Use the non-breaking space (nbsp) 4 times to insert a tab
  • Use “br” to add a line break
